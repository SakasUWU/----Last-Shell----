# 🎮 Last Shell — Launcher

Descarga e instala **Last Shell** automáticamente.

## ⬇️ Descargar

Descarga el **último `last-shell-launcher.exe`** de las [Releases](https://github.com/SakasUWU/Last-Shell-Launcher/releases)

## 🚀 Instalar

1. Descarga `last-shell-launcher.exe`
2. Ejecuta el archivo
3. ¡Automáticamente descarga el juego actualizado!

## ✨ Características

- ✅ Auto-actualización a la última versión
- ✅ Funciona sin internet (lanza versión local)
- ✅ Backup automático de versiones anteriores
- ✅ Windows + Linux compatible

## 📝 Uso

```bash
last-shell-launcher.exe
```

**Primera vez:** Descarga el juego (~50-100 MB)
**Siguientes veces:** Revisa actualizaciones automáticamente
